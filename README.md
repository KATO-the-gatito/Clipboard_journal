# О программе
Журнал буфера обмена позволяет хранить всё, что копировал пользователь. Любой скопированный текст (запись) сразу добавляется в журнал и сохраняется там до тех пор, пока пользователь не удалит его или не завершит работу программы. По умолчанию программа работает в фоне, но её всегда можно вызвать, чтобы увидеть всё ранее скопированное. Рекомендуется добавить эту программу в автозапуск.

# Горячие клавиши
***Когда окно свернуто:***
1) **Ctrl + Alt + V** - вызвать журнал

***Когда окно активно:***
1) **Esc** - закрыть журнал 
2) **Enter** - вставить запись
3) **Del** - удалить запись
4) **Up/Down** - навигация по записям
5) **Left/Right** - навигация по буферам
6) **M** - добавить запись в "помеченные" (Marked data)
